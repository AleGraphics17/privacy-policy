Política de Privacidad - Shader Realistic

Desarrollado por: Ale Studio
Última actualización: 29 de Julio, 2024
Versión de la aplicación: 1.0.0
⚠️ Importante: Esta aplicación está dirigida a usuarios de todas las edades, incluyendo menores de 13 años. Cumplimos estrictamente con las leyes de protección de datos infantiles.

Información que Recopilamos

Información Automática

  • Datos de uso: Información sobre cómo usa la aplicación
  • Información del dispositivo: Tipo de dispositivo, sistema operativo, identificadores únicos
  • Datos de anuncios: Google AdMob recopila datos para mostrar anuncios personalizados
  • Datos de rendimiento: Información sobre el funcionamiento de la aplicación

Información que NO Recopilamos

  • Información personal identificable (nombre, email, teléfono)
  • Información de ubicación precisa
  • Información de contacto
  • Datos biométricos
  • Información financiera
  • Contenido de comunicaciones personales

Protección Especial para Menores de 13 Años

Compromiso con la Protección Infantil

  • No recopilamos información personal de menores de 13 años
  • No solicitamos información personal a niños
  • Los anuncios mostrados son apropiados para todas las edades
  • No compartimos datos de menores con terceros

Anuncios Apropiados para Menores

  • Utilizamos AdMob con configuración de contenido apropiado para menores
  • Los anuncios no contienen contenido inapropiado
  • No se recopilan datos de menores para publicidad personalizada

Cómo Usamos la Información

Para Proporcionar Servicios

  • Mostrar anuncios relevantes y apropiados
  • Mejorar la funcionalidad de la aplicación
  • Resolver problemas técnicos
  • Optimizar el rendimiento de la aplicación

Para Anuncios

  • Google AdMob utiliza datos para mostrar anuncios apropiados
  • Los anuncios pueden basarse en el tipo de dispositivo y uso básico
  • No se utiliza información personal para publicidad dirigida

Compartir Información

Terceros

  • Google AdMob: Para mostrar anuncios apropiados
  • Google Analytics: Para análisis de uso básico (si se implementa)

No Vendemos Información

No vendemos, alquilamos o compartimos su información personal con terceros para fines comerciales. Especialmente, nunca compartimos información de menores de 13 años.

Seguridad

Implementamos medidas de seguridad razonables para proteger su información, pero ningún método de transmisión por internet es 100% seguro. Utilizamos protocolos de seguridad estándar para proteger los datos.

Sus Derechos

Puede:

  • Solicitar acceso a sus datos
  • Solicitar corrección de datos inexactos
  • Solicitar eliminación de datos
  • Optar por no recibir anuncios personalizados
  • Retirar el consentimiento en cualquier momento

Derechos de los Padres

Si es padre o tutor de un menor de 13 años que usa nuestra aplicación:

  • Puede solicitar la eliminación de cualquier información recopilada
  • Puede optar por no permitir la recopilación de datos
  • Puede contactarnos para cualquier consulta sobre la privacidad de su hijo

Almacenamiento de Datos

  • Duración: Los datos se almacenan solo mientras sea necesario para proporcionar el servicio
  • Ubicación: Los datos se procesan en servidores seguros
  • Eliminación: Los datos se eliminan automáticamente cuando ya no son necesarios

Cambios a esta Política

Podemos actualizar esta política ocasionalmente. Los cambios importantes se notificarán a través de la aplicación. Para cambios que afecten a menores de 13 años, se requerirá consentimiento parental.

Información Específica de AdMob

Esta aplicación utiliza Google AdMob para mostrar anuncios. AdMob puede recopilar y procesar datos según su propia política de privacidad disponible en: https://policies.google.com/privacy

Datos Recopilados por AdMob

  • Identificadores de dispositivo (no personales)
  • Información de uso básica de la aplicación
  • Datos de rendimiento de anuncios
  • Tipo de dispositivo y sistema operativo

Optar por No Recibir Anuncios Personalizados

Puede optar por no recibir anuncios personalizados en la configuración de su dispositivo o a través de las opciones de AdMob.

Información Específica de la Aplicación

Shader Realistic para Minecraft

Esta aplicación permite descargar e instalar shaders realistas para Minecraft Bedrock Edition. La aplicación:

  • No recopila información personal
  • No requiere registro de usuario
  • Solo descarga archivos de shaders a su dispositivo
  • Muestra anuncios apropiados para mantener la aplicación gratuita
  • Es segura para usuarios de todas las edades

Permisos Utilizados

  • Almacenamiento: Para guardar archivos de shaders descargados
  • Internet: Para cargar anuncios y descargar contenido
  • Estado de red: Para verificar conectividad

Datos del Dispositivo

La aplicación puede acceder a:

  • Información básica del dispositivo (modelo, sistema operativo)
  • Espacio de almacenamiento disponible
  • Estado de conexión a internet

Esta información se utiliza únicamente para:

  • Optimizar la experiencia de descarga
  • Mostrar anuncios apropiados
  • Resolver problemas técnicos
  • Mejorar el rendimiento de la aplicación

Cumplimiento Legal

  • GDPR: Respetamos los derechos de privacidad de usuarios europeos
  • CCPA: Cumplimos con la Ley de Privacidad del Consumidor de California
